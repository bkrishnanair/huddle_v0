# Git Commands for Huddle Project

## Standard Development Workflow
```bash
# Add all changes
git add .

# Commit with descriptive message
git commit -m "fix: resolve Next.js 15 authentication compatibility issues"

# Push to main branch
git push origin main
```

## Common Git Operations
```bash
# Pull latest changes
git pull origin main

# Create and switch to new feature branch
git checkout -b feature/new-functionality

# Merge feature branch (after switching to main)
git merge feature/new-functionality

# Check current status
git status

# View commit history
git log --oneline

# View differences
git diff
```

## Branch Management
```bash
# View all branches
git branch -a

# Switch to main branch
git checkout main

# Delete merged feature branch
git branch -d feature/branch-name
```
